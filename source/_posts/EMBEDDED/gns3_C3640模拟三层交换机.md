**gns3对支持三层交换机并不完善。但也可以模拟。**

<!-- more -->
以C3640为例，需要添加NM-16ESW交换模块端口。

添加vlan需在vlan database模式下。

查看为show vlan-switch

如果交换端口直接连接的为PC，

则需要在端口模式下配置

speed 100

duplex full

不然端口是不能全up的。

这个问题还真是让我愁了一上午，额。

命令如下:


```
enable
vlan database
vlan 2
exit
configure terminal
interface range fastethernet0/2 -17
switchport mode access
switchport access vlan 1
exit

```
